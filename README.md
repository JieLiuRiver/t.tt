###安装
***
项目地址：（`git clone`）
```shell
git clone https://github.com/stupidWall/t.tt.git
```
通过`npm`安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))

```
npm install
```
启动服务(http://localhost:8088)

```
npm run dev
```
发布代码

```
npm run build
```

## 页面数据
没有异步获取，直接存在bus.js文件中，没有使用vuex，使用中央事件派发bus代替
